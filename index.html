Etiquetas Básicas

<!DOCTYPE html>

Indica al navegador que el documento está escrito en HTML5.

<!DOCTYPE html>

<html>

Es la etiqueta raíz de todo documento HTML.

<html>...</html>

<head>

Contiene metadatos del documento.

<head>...</head>

<body>

Contiene el contenido visible de la página.

<body>Contenido</body>

Tipos de Inputs en HTML

<input type="text">

Permite al usuario escribir texto corto como nombres o datos básicos.

¿Para qué sirve? Se usa para capturar información escrita por el usuario.

<input type="text" placeholder="Escribe tu nombre">

<input type="email">

Permite ingresar una dirección de correo electrónico.

¿Para qué sirve? El navegador valida automáticamente el formato del email.

<input type="email" placeholder="correo@email.com">

<input type="password">

Permite ingresar contraseñas ocultando los caracteres.

¿Para qué sirve? Se usa para proteger información sensible.

<input type="password" placeholder="Contraseña">

<input type="number">

Permite ingresar únicamente números.

¿Para qué sirve? Se usa para datos numéricos como edad o cantidades.

<input type="number" min="1" max="100">

<input type="date">

Permite seleccionar una fecha usando un calendario.

¿Para qué sirve? Se usa para registrar fechas como cumpleaños o reservas.

<input type="date">

<input type="color">

Permite seleccionar un color mediante un selector visual.

¿Para qué sirve? Se usa para personalizar colores en aplicaciones.

<input type="color">

<input type="file">

Permite subir archivos desde el dispositivo.

¿Para qué sirve? Se usa para cargar documentos, imágenes o archivos.

<input type="file">

<input type="checkbox">

Permite seleccionar varias opciones.

¿Para qué sirve? Se usa cuando el usuario puede elegir múltiples opciones.

<input type="checkbox"> HTML <input type="checkbox"> CSS <input type="checkbox"> JavaScript HTML CSS JavaScript

<input type="radio">

Permite seleccionar solo una opción dentro de un grupo.

¿Para qué sirve? Se usa cuando el usuario debe elegir solo una respuesta.

<input type="radio" name="lenguaje"> HTML <input type="radio" name="lenguaje"> CSS <input type="radio" name="lenguaje"> JavaScript HTML CSS JavaScript

<input type="range">

Permite seleccionar un valor dentro de un rango usando un control deslizante.

¿Para qué sirve? Se usa para niveles como volumen o intensidad.

<input type="range" min="0" max="10">

<textarea>

Permite ingresar textos largos.

¿Para qué sirve? Se usa para comentarios o mensajes.

<textarea rows="4"></textarea>
